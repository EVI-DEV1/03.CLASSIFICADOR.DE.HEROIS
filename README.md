Desafios de Lógica com JavaScript

Este repositório reúne os três desafios de lógica desenvolvidos durante meus estudos em linguagem de programação, colocando em prática conceitos fundamentais da programação e utilizando JavaScript.

 📚 Conteúdos praticados
- Variáveis
- Operadores
- Estruturas de decisão
- Laços de repetição
- Funções
- Classes e Objetos

 🚀 Projetos desenvolvidos
1️⃣ Classificador de Heróis
Sistema que classifica o nível de um herói com base na quantidade de XP.
2️⃣ Calculadora de Partidas Rankeadas
Projeto que calcula o saldo de vitórias e determina o nível do jogador.
3️⃣ Classes de um Jogo
Criação de uma classe genérica de herói com diferentes tipos de ataque utilizando Programação Orientada a Objetos (POO).

🛠️ Tecnologias utilizadas
- JavaScript
- Node.js
- VS Code

🎯 Objetivo
Praticar lógica de programação e fortalecer a base em JavaScript através de desafios práticos.
